# XXHash3.NET

Pure C# implementation of the XXHash3 algorithm

# Features

- XXHash3 64-bit
  - Oneshot hashing API
  - Streaming hashing API

## Planned

- XXHash3 128-bit
- XXHash32
- XXHash64
- Implement SSE2/AVX boosted operations
